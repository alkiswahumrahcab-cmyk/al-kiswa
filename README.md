# Al Kiswah Umrah Transport

This is the new repository for the Al Kiswah Umrah Transport project.

## Setup

1.  **Database**: Create a `.env` file in the root directory.
    ```env
    DATABASE_URL="your_mongodb_connection_string"
    NEXT_PUBLIC_APP_NAME="Al Kiswah Umrah Transport"
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run development server**:
    ```bash
    npm run dev
    ```
